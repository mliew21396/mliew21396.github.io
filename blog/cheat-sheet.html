Ruby vs JavaScript

August 5th, 2015

Here's a quick cheatsheet on comparing Ruby vs JavaScript. Paulo and I worked together on this and listed the most common differences below. We also found a really Amazing Resource that goes more indepth on this topic. Check it out!

Ruby

JavaScript

Method/Function Definitions

#Defines function
def method_name (parameter1, parameter2)
  puts parameter1
end

# Calls function
method_name("cat", "dog") # => cat
        
//Defines function
def methodName (parameter1, parameter2) {
  console.log(parameter1);
};

// Calls function
methodName("cat", "dog") // => cat

Variable Declarations

#Declares Variable
variable_name = "cat"
        
//Declares Variable
var variableName = "cat";

Create Class/Object

#Ruby Class
class Person
  attr_accessor :firstName, :lastName

  def initialize(firstName, lastName)
    @firstName = firstName
    @lastName = lastName
  end

  def fullName
    @firstName + " " + @lastName
  end
end

john = Person.new("John", "Malkovic")
john.fullName # "John Malkovic"
        
//JavaScript Class
function Person(firstName, lastName) {
  this.firstName = firstName;
  this.lastName = lastName;
}




Person.prototype.fullName = function() {
  return this.firstName + " " + this.lastName;
}


var john = new Person("John", "Malkovic");
john.fullName(); // "John Malkovic"

Hashes and Objects

#Declares Variable
h = {}
h['a'] = 1
h['b'] = 2

h.each {|key, value| puts "#{key} #{value}" }

h.keys # ['a', 'b']
h.has_key?('c') # false

h.length # 2

h.delete("b")
        
//Declares Variable
var h = {};
h['a'] = 1;
h['b'] = 2;

for (key in h) { console.log(key, h[key]); }

Object.keys(h); // ['a', 'b']
h.hasOwnProperty('c') // false

Object.keys(h).length // 2

delete h.b
